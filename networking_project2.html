Networking Concepts

Key networking concepts connected to our Project 2 website implementation.

DNS

DNS translates human-readable domain names into IP addresses. When a user visits a website, DNS resolves the domain name into the server address.

IP Addressing

Every device connected to the internet uses an IP address. Websites can support IPv4 and IPv6 so modern networks can reach them.

HTTP vs HTTPS

HTTP transfers web pages without encryption. HTTPS adds TLS encryption, which protects website traffic from interception and tampering.

CDN

A Content Delivery Network stores website assets closer to users. This can reduce latency and improve load times.

Caching

Caching stores copies of files so repeat visits load faster and reduce unnecessary network requests.

Monitoring

Monitoring tools track traffic, performance, errors, and suspicious behavior so developers can detect issues faster.
